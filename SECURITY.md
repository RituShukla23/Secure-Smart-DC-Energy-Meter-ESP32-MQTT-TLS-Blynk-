# Security Policy

## Supported Content

This repository is intended for an academic smart energy meter project.

## Reporting Issues

If you find a security issue, such as:

- a hardcoded secret,
- an insecure TLS configuration,
- or an exposed lab artifact,

document it privately before publishing the repository.

## Security Guidance

- Replace placeholder Wi-Fi and Blynk values before building.
- Use certificate-validated TLS for production deployments.
- Do not commit packet captures that contain live credentials.
- Review images and PDFs before publishing if they contain private network or account information.
