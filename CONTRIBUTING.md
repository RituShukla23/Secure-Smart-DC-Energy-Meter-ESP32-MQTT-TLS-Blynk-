# Contributing

This repository is maintained as a project showcase for a secured smart DC energy meter.

## Before You Contribute

- Keep credentials, tokens, and private network details out of version control.
- Prefer placeholders or environment-specific configuration files that are ignored by Git.
- Keep edits focused and documented.

## Recommended Workflow

1. Create a feature branch.
2. Make the smallest useful change.
3. Verify the sketch or documentation locally.
4. Open a pull request with a short summary of the change and any security impact.

## Scope

Good contributions include:

- Documentation improvements.
- Security hardening.
- Build or deployment guidance.
- Refactoring that improves clarity without changing behavior.

## Not Accepted

- Raw secrets.
- Unredacted packet captures.
- Large unrelated formatting-only changes.
